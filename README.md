# I2C Master in Verilog

I2C Master implemented in Verilog and simulated using Vivado 2021.2.

## Features

* I2C start condition generation
* I2C stop condition generation
* 7-bit slave address transmission
* 8-bit data transmission
* Behavioral simulation
* Testbench verification
* Waveform analysis

## Tools

* Verilog
* Vivado 2021.2
* Arty A7-100 FPGA board

* i2c_master.v
